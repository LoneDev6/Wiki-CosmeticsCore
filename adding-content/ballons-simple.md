### Step 2

Create the cosmetic configuration

```yaml
  simple_diamond_block_balloon:
    enabled: true
    display_name: "Simple Diamond Block Balloon"
    type: BALLOON
    model:
      gui: DIAMOND_BLOCK
      normal: DIAMOND_BLOCK
```

You're done
