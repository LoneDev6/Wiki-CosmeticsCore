### Step 2

Create the cosmetic configuration

```yaml
  simple_itemsadder_item_balloon:
    enabled: true
    display_name: "Simple ItemsAdder Item Balloon"
    type: BALLOON
    model:
      gui: iadeco:old_clock
      normal: iadeco:old_clock
```

You're done
