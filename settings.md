## Enable/disable in worlds

Enable it in particular worlds:

```yml
cosmetics:
  worlds:
    - "world"
    - "world_nether"
```

Enable it in all worlds:

```yml
cosmetics:
  worlds:
    - "*"
```

Enable it in worlds which name starts with `world_` :

```yml
cosmetics:
  worlds:
    - "world_*"
```
